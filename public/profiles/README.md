# Profilfotos

Lege hier **6 Bilder** ab (JPG oder PNG), z. B.:

- `1.png` … `6.png`

Die Dateinamen müssen zu `src/assets/data.ts` passen (`profileImagePath('1.png')` usw.).

**Empfehlung:** quadratisch, mindestens 400×400 px, Gesichter gut sichtbar.

Nach dem Ersetzen: `npm run dev` neu laden oder neu deployen.
