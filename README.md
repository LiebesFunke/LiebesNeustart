# Zeit für Liebe

Landing (DE) mit Quiz und Partner-Link — React, Tailwind, Framer Motion.

## Schnellstart

```bash
npm install
npm run dev
```

## Anpassen

| Was | Wo |
|-----|-----|
| Partner-Link | `src/config/site.ts` → `partnerUrl` |
| Profilfotos | `public/profiles/1.jpg` … `6.jpg` |
| Texte | `src/assets/data.ts` |
